# Installation

## Requirements

- **Blender 2.79** (the addon is not compatible with Blender 2.8 or later due to API differences)
- A modern mobile browser with WebXR support (Chrome on Android, Safari 15+ on iOS)

---

## Steps

### 1. Download the addon

Download the `range_webar_exporter.zip` file. **Do not unzip it** — Blender installs directly from the zip.

### 2. Open Blender Preferences

Go to **File → User Preferences** (or press `Ctrl+Alt+U`).

### 3. Install the zip

1. Click the **Add-ons** tab.
2. Click **Install Add-on from File…** at the bottom.
3. Navigate to and select `range_webar_exporter.zip`.
4. Click **Install Add-on from File**.

### 4. Enable the addon

After installing, search for **"Range WebAR"** in the add-on list and check the checkbox to enable it.

### 5. Save preferences

Click **Save User Settings** so the addon stays enabled after restarting Blender.

---

## Verify the installation

Open the **3D View**, press `N` to open the sidebar (or look for a **WebAR** tab on the right side of the 3D View). If you see the **WebAR Exporter** panel, the addon is installed correctly.

!!! tip "Can't find the panel?"
    Make sure you are in the **3D View** editor, not the Node Editor or UV editor. The panel lives in **View3D > Sidebar > WebAR**.

---

## Uninstalling

Go to **File → User Preferences → Add-ons**, search for "Range WebAR", expand the entry, and click **Remove**.
